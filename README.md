#Exploding Cat Project
This is the repository for the Exploding Cat project, a multiplayer card game developed using React, Node.js, and MongoDB.

#Description
Exploding Cat is a card game designed for multiplayer interactions. Players take turns drawing cards from a deck, with the goal of avoiding exploding cats while using various cards to skip turns, shuffle the deck, or counter other players' actions.

#Features
Multiplayer functionality
Card drawing and gameplay mechanics
Deck shuffling and card effects
User authentication and authorization
MongoDB integration for data storage
#Technologies Used
React.js
Node.js
Express.js
MongoDB
Socket.io (for real-time communication)
HTML/CSS (for UI)
